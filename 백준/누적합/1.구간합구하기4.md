# 문제 (실버 3)

수 N개가 주어졌을 때, i번째 수부터 j번째 수까지 합을 구하는 프로그램을 작성하시오.

링크 - [https://www.acmicpc.net/problem/11659](https://www.acmicpc.net/problem/11659)

# 입력과 출력

```
5 3
5 4 3 2 1
1 3
2 4
5 5
```

총 M개의 줄에 입력으로 주어진 i번째 수부터 j번째 수까지 합을 출력한다.

```
12 (5+4+3) -> psum[3]
9 (4+3+2) -> psum[4] - psum[1]
1 (1) -> psum[5] - psum[4]
```
